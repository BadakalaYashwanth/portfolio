# Badakala Yaswanth — Developer Portfolio

> **Live Site:** [https://mydevportolio.vercel.app/](https://mydevportolio.vercel.app/)

A high-performance, modern developer portfolio built with **React + Vite**, featuring smooth animations, a Ferrari-inspired dark theme, and a fully responsive layout.

---

## 🚀 Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 19 |
| Build Tool | Vite 5 |
| Styling | Tailwind CSS |
| Animations | Framer Motion |
| Icons | Lucide React |
| Fonts | Inter · Space Grotesk · JetBrains Mono |
| Deployment | Vercel |

---

## ✨ Features

- **Ferrari-inspired dark theme** with custom color tokens
- **Smooth scroll animations** powered by Framer Motion
- **Fully responsive** — works on mobile, tablet, and desktop
- **Sections:** Hero · About · Projects · Experience · Skills · Process · Achievements · Certifications · Contact
- **Copy-to-clipboard email** with animated feedback
- **Premium custom scrollbar** styling
- **SEO optimized** with meta tags and semantic HTML

## 🛠️ Getting Started

### Prerequisites
- Node.js 18+
- npm

### Install & Run

```bash
# Clone the repository
git clone https://github.com/BadakalaYashwanth/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder.
